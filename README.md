# Circle Connect

This project is a React application built with Vite.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
npm install
```

### Development

Start the development server:

```sh
npm run dev
```

### Build

Build for production:

```sh
npm run build
```

## Technologies

- Vite
- React
- TypeScript
- shadcn-ui
- Tailwind CSS
